### Crea un directorio para el proyecto
1. Crea un nuevo directorio el cual se usará como área de trabajo.
```
mkdir shrek_app
```
![creacion_directorio](./Images/1.png)

2. Accede al directorio creado en el paso anterior.
```
cd shrek_app
```
